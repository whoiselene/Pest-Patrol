# 🚁 Pest Patrol

**AI-powered drone system for agricultural pest detection and plant health monitoring.**

---

## Overview

Pest Patrol is a sustainability-focused project that puts machine learning in the sky. Drones fly over farmland, capture live aerial footage, and an on-board computer vision pipeline instantly classifies whether plants are healthy or under threat — giving farmers actionable insight before damage spreads.

Built at the intersection of drone engineering, computer vision, and deep learning, Pest Patrol was developed to make precision agriculture accessible and reduce crop loss through early detection.

---

## How It Works

```
1. Drone captures aerial footage
        ↓
2. OpenCV detects plant regions in frame
        ↓
3. ROI (Region of Interest) extracted
        ↓
4. CNN model classifies plant health
        ↓
5. Health prediction output generated
        ↓
6. Real-time visualization displayed
```

## Features

| Feature | Description |
|---|---|
| 🚁 **Drone-Assisted Monitoring** | Aerial image collection over farmland in real time |
| 👁️ **Computer Vision Pipeline** | OpenCV-based plant region detection |
| 🧠 **CNN Classification** | Deep learning model distinguishing healthy vs unhealthy crops |
| ⚡ **Real-Time Analysis** | Live visual data processed on the fly |
| 🌱 **Sustainability Focus** | Reduces pesticide overuse by targeting problem areas precisely |

## Why It Matters

Traditional pest monitoring relies on manual field inspection — slow, expensive, and often too late. Pest Patrol automates that process from above, covering more ground in minutes than a farmer could in days. By catching unhealthy crops early and precisely, it helps reduce chemical intervention, cut losses, and move agriculture toward a more sustainable future.
